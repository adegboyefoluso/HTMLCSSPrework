Position

The position property specifies the type of positioning method used for an element. The four position values are :

  • fixed:this will pin something to a spot, and it ,will stay there as you scrol.(ex: nav bar) Fixed position takes elements out of the normal document flow.
  • Static:this is the default setting, it is not affected by top botoom, left and right properties. It is always postioned according to the normal flow of the page
  • relative:does not take element out of thier normal document flow. still appears
    within its space/div/container, but allows it to be offset a bit from the top, bottom, right left .
    Top right bottom and left properties will move it away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
  • absolute:takes element out of the normal document flow , If a parent element position ,is set to relative, and the child element is set to absolute
    the child element's new(0,0) cordionate ,will be the upper left hand corner of the parent element instead of the whole page.

The z-indexproperty alows yoiu to override the normal stacking order of the page.
tHe default is 0, and you can set that value to be positive or negative.
An element with a greater z-index will stack in fromn of an element with a lower z-index . if two elements have the same z-index, or don't have one assigned, the element
postioned last ,in the HTML will show on top. You must have a position assigned in order to assign a z-index.
